## Features

- **Responsive Design** — works seamlessly on mobile, tablet, and desktop
- **Typing Animation** — dynamic role cycling on the hero section
- **Scroll Reveal** — smooth entrance animations as you scroll
- **Parallax Effect** — subtle depth on the featured section
- **Project Showcase** — interactive cards with live links and tech tags
- **Contact Form** — powered by [Formspree](https://formspree.io)
- **Back to Top** — quick navigation utility
